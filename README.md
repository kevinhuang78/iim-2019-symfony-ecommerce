# IIM A4 IWM Symfony 4 E-Commerce (Class)

## Dependencies

- Curl
- Composer
- PHP (At least 7.1)

## Install the project

- `git clone https://github.com/kevinhuang78/iim-2019-symfony-ecommerce.git`
- `composer install`

## Configure

- Change `.env` to connect database

## Run the project

- `php bin/console server:run`
